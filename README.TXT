PROJECT TITLE: MALCardNamer
PURPOSE OF PROJECT: To facilitate creating an Adobe Photoshop script and running it without
scripting knowledge by using drag and drop functionality created through java swing components
VERSION or DATE: 9/1/21
HOW TO START THIS PROJECT: Run it's executable
AUTHORS: Vandell Vatel
USER INSTRUCTIONS:
Step 1: Run the executable (.exe)
Step 2: Find your adobe photoshop application, 
Step 3: Add your psds, by doing File->Open PSDs,
Step 4: Add users either by selecting them from the search bar, or typing in their name and hitting enter.
Step 5: Drag and drop the psds that you opened onto the each user's lists
Step 6: When ready, hit save, choose where to save the images, and watch the magic happen.
